# esp32_joystick_controller
A basic <strong>esp32 joystick controller</strong> that connects with different bots via wifi or espnow if available,<br> this is just a basic take of mine for robotics journey.

<h5>yt playthrough (RuneRadiance): https://www.youtube.com/watch?v=6VAcHWxVyPE</h5>

<strong>NOTE: This project has been done using micropython and Thonny IDE and uses ESPNOW comm where both the controller and the bot needs to use ESP32</strong><br>
<p>
1. Download the RR01-multipurpose nrf remote (ESP32 30pin dev board).py file<br>
2. Rename it as 'main.py' [in the case of ESP32]<br>
3. Upload the main.py file onto your controlller and make changes as needed for robot's mac and bot list<br>
4. Use another esp32 on ur robot and u can code it to connect with espnow and enjoy<br>
5. Also, you can remap the esp32 pins or tweak the code as u wish, it's upto you!
</p>
