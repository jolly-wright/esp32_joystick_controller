# ESP32 MULTIPURPOSE JOYSTICK CONTROLLER
A basic <strong>esp32 joystick controller</strong> that connects with different bots via wifi or espnow if available,<br> this is just a basic take of mine for robotics journey.

<strong>NOTE: This project has been done using micropython and Thonny IDE and uses ESPNOW comm where both the controller and the bot needs to use ESP32</strong>

1. <a href='https://www.youtube.com/watch?v=6VAcHWxVyPE'>Watch the tutorial</a>
2. Also make sure to check the circuit schematic for wiring and connection
3. Download the RR01-multipurpose nrf remote (ESP32 30pin dev board).py file
4. Rename it as 'main.py' [in the case of ESP32]
5. Upload the main.py file onto your controlller and make changes as needed for robot's mac and bot list
6. Use another esp32 on ur robot and u can code it to connect with espnow and enjoy
